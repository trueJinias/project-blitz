
# Affiliate & Image Guidelines

## 1. Product Images
- **Source Specific Images:** Do not use generic images (e.g. "white t-shirt" from Pexels).
- **Search Strategy:**
  1.  **Manufacturer/Retailer:** Try official sites or wholesalers (e.g. Philips, Hanes).
  2.  **YouTube Thumbnails:** Use `https://img.youtube.com/vi/[VIDEO_ID]/maxresdefault.jpg` for review videos. This avoids hotlinking blocks.
  3.  **Fallback:** Only use high-quality Unsplash/Pexels if specific images are impossible to get, but rename them to match the product (e.g. `panasonic-steamer.jpg`).
- **NO HOTLINKING:** Never link to Amazon (`m.media-amazon.com`) directly. Download locally.
- **Verification:** Ensure file > 3KB.

## 2. Affiliate Links
- **Search Links ONLY:**
  - **JP:** `https://www.amazon.co.jp/s?k=[KEYWORDS]&tag=blitz011-22`
  - **US:** `https://www.amazon.com/s?k=[KEYWORDS]&tag=blitz011-20`
  - **IN:** `https://www.amazon.in/s?k=[KEYWORDS]&tag=blitz011-21`
- **Rakuten:** `https://search.rakuten.co.jp/search/mall/[KEYWORDS]/`

## 3. SNS Generation
- **Full Text:** Always verify the generated tweet text is complete.
- **Intent URL:** Ensure the intent URL is clickable and includes the text.
