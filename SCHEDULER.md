# 記事自動生成スケジューラー セットアップガイド

## 概要
朝9時〜夜8時の間、1時間ごとに `/article-generator` コマンドを自動送信するAutoHotkeyスクリプト。

## セットアップ手順

### 1. スクリプトを起動
```
scripts\article-scheduler.ahk をダブルクリック
```

### 2. VS Codeの準備
- VS Codeを開いておく
- Geminiチャット画面を表示した状態にする

### 3. 自動実行開始
- 確認ダイアログが表示されたら「Yes」で即時実行
- 以降は1時間ごとに自動実行

## 設定変更

`scripts\article-scheduler.ahk` を編集：

```ahk
INTERVAL_MS := 3600000  ; 実行間隔（ミリ秒）
START_HOUR := 9         ; 開始時刻
END_HOUR := 20          ; 終了時刻
```

| 間隔 | 設定値 |
|------|--------|
| 30分 | 1800000 |
| 1時間 | 3600000 |
| 2時間 | 7200000 |

## 操作方法

| 操作 | 方法 |
|------|------|
| 手動実行 | `Ctrl+Alt+A` |
| 停止 | タスクトレイの H アイコン → Exit |
| 再開 | スクリプトをダブルクリック |

## スタートアップ登録（PC起動時に自動開始）

1. `Win+R` で「ファイル名を指定して実行」を開く
2. `shell:startup` と入力してEnter
3. 開いたフォルダに `article-scheduler.ahk` のショートカットを作成
